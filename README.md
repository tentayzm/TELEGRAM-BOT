## 🚀 نصب و راه‌اندازی

### 1. پیش‌نیازها
```bash
pip install -r requirements.txt
```

### 2. تنظیمات
فایل `config.py` را باز کنید و موارد زیر را تنظیم کنید:

```python
# توکن ربات از @BotFather
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"

# آیدی عددی سودو (مدیر اصلی ربات)
SUDO_USERS = [123456789]  # آیدی خودتان
```

### 3. اجرا
```bash
python main.py
```

## 🔐 نکته مهم

> ⚠️ **فقط سودو ربات می‌تواند ربات را به گروه اضافه کند!**
> 
> اگر کسی غیر از سودو ربات را به گروه اضافه کند، ربات به طور خودکار از گروه خارج می‌شود.

## 📝 متغیرهای پیام خوش‌آمد

در پیام خوش‌آمدگویی و خداحافظی می‌توانید از این متغیرها استفاده کنید:

| متغیر | توضیحات |
|--------|---------|
| `{mention}` | منشن کاربر |
| `{first_name}` | نام کاربر |
| `{last_name}` | نام خانوادگی |
| `{full_name}` | نام کامل |
| `{username}` | یوزرنیم |
| `{user_id}` | آیدی عددی |
| `{group}` | نام گروه |
| `{chat_id}` | آیدی گروه |

## 🔧 توسعه

برای اضافه کردن قابلیت جدید:

1. هندلر را در `handlers.py` یا `handlers2.py` بنویسید
2. دستور را در `main.py` ثبت کنید
3. در صورت نیاز کیبورد را در `keyboards.py` اضافه کنید
